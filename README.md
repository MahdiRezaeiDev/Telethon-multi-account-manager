# Telethon Multi-Account Manager

A Python-based Telegram multi-account management system built with [Telethon](https://docs.telethon.dev/).
This project allows you to register multiple Telegram accounts, listen to group messages, and send automated responses, including price checks or other data, all using account sessions.

---

## Features

- ✅ Register multiple Telegram accounts and store sessions.
- ✅ Two-step login support including password verification.
- ✅ Download and store Telegram profile photos.
- ✅ Listen to messages from specific Telegram groups or channels.
- ✅ Automatically process incoming messages, check codes or usernames, and reply with price data.
- ✅ Handle privacy restrictions and notify account owners if a message could not be delivered.
- ✅ MySQL integration for storing accounts, messages, and price data.
- ✅ Async and multi-account support using `asyncio` and Telethon.
